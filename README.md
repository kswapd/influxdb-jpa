# TSDB-EXPRESS

-----------
##Introduction
A time series database middleware to connect influxdb.

##Requirement

*  `JDK version 1.7 or later`
*  `maven version 3.0+`
* `Spring Framework 4.0.0 or later`


###Compile

 `mvn clean install`
 
###maven import
```
<dependency>
        <groupId>dcits</groupId>
        <artifactId>tsdb-express</artifactId>
        <version>1.0-SNAPSHOT</version>
</dependency>
```
